Modules to install:
    Pygame
1. Run the server first and input the IP Address and port you want to use in the terminal when prompted
2. Run the client file click in the window and input the server IP and Port in the terminal when prompted
3. Game will not start until until two players are connected so repeat step 2 with a new client instance
4. Select which move to make by clicking the box towards the bottom
5. To quit click the the X on the top right corner
-If one client is still connected when the other one disconnects the remaining player will return to the menu to reconnect
